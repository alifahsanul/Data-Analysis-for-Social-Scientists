# Data Analysis for Social Scientists — personal notes

This repository holds my personal study materials for **data analysis in the social sciences**. I am using it to learn concepts, methods, and practice—not as an official course submission.

## How I learn

I use **AI as a study partner**: I feed it lecture materials (transcripts, slides, and homework) and ask it to help me understand, organize, and summarize the content.

## Tools

- **Obsidian** — my main reading and navigation surface; I browse and link notes here.
- **Cursor** — where I work with the AI to draft and refine **Markdown** notes that end up in this vault.

Together, Obsidian is the front end for consuming notes; Cursor is where most of the Markdown is generated and edited with AI assistance.
